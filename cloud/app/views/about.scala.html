Getting data from SicsthSense

You can fetch data from SicsthSense's database through a GET at the following URI: 
  http://sense.sics.se/stream/[user]/[device]/[path]
URI arguments allow to get a subset of the dataset. Use ?tail=n to get the n most recent samples Use ?last=t to get samples posted during the last t seconds. Use ?since=t to get the samples posted since (including) absolute UNIX time t (in seconds).

Accessing devices through SicsthSense

SicsthSense can act as a RESTful proxy towards any public device. This allows yu to share your devices without making their URI public. Just GET, POST, PUT or DELETE to: 
  http://sense.sics.se/proxy/[user]/[device]/[path]

Resource discovery

Devices can provide a description of the resources they offer. This description must be hosted at /discover, and have the following format: 
  {
    "uid": "uid",
    "resources": [
      "/sensors/temperature",
      "/sensors/energy"
    ]
  }
